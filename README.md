# SIMPLAY

## REQUIREMENTS
### GRUNT-CLI
	$ npm install grunt-cli -g
### BOWER
	$ npm install bower -g
### SASS
	$ gem install sass


## HOW TO USE
### The first time
	$ bower install                         
	$ npm install                          
	$ grunt                                  
### Working on the project                                 
	$ grunt dev
	                   
### When work is finished
	$ grunt prod
